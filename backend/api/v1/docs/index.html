1. API Overview

The GTV Motor API v1 provides programmatic access to customer, vehicle, invoice, and analytics data. This RESTful API uses API key authentication and returns JSON responses.

Key Features

  • ‚úÖ RESTful API design with standard HTTP methods
  • ‚úÖ API key-based authentication
  • ‚úÖ Comprehensive customer and vehicle management
  • ‚úÖ Service invoice tracking
  • ‚úÖ Real-time analytics and usage statistics
  • ‚úÖ Pagination support for large datasets
  • ‚úÖ Search and filtering capabilities
  • ‚úÖ Request analytics and monitoring
‚ÑπÔ∏è API Version: v1
Base URL: https://api.gtvmotor.dev/api/v1
Content-Type: application/json
Response Format: JSON

2. Authentication

All API requests require authentication using an API key. The API key must be included in the request header.

API Key Header

Include your API key in the X-API-Key header for every request:

X-API-Key: your-api-key-here

Getting an API Key

API keys can be obtained through the API key management endpoint or by contacting your administrator. Each API key has specific permissions:

  • read - Allows GET requests to retrieve data
  • write - Allows POST, PUT, PATCH requests to create/update data
  • admin - Full access including DELETE operations
‚ö†Ô∏è Security Note: Never expose your API key in client-side code or public repositories. Always keep your API keys secure and rotate them regularly.

4.5 Analytics

Endpoint: GET /analytics

Get API usage statistics and analytics data.

Query Parameters

Parameter Type Required Description
type string Optional Analytics type: overview, endpoints, keys, errors, performance, traffic (default: overview)
days integer Optional Number of days to analyze (default: 7)
breakdown string Optional Breakdown period: day, month, year (default: day)
endpoint string Optional Filter by specific endpoint

Analytics Types

  • overview - Overall statistics and trends
  • endpoints - Statistics per endpoint
  • keys - API key usage statistics
  • errors - Error analysis and breakdown
  • performance - Performance metrics by hour
  • traffic - Traffic overview (last 24 hours)

Example Requests:

  • GET /analytics?type=overview&days=7&breakdown=day - Daily overview for last 7 days
  • GET /analytics?type=overview&days=365&breakdown=month - Monthly overview for last year
  • GET /analytics?type=endpoints&days=30 - Endpoint statistics for last 30 days
  • GET /analytics?type=keys&days=7 - API key usage for last 7 days

5. Request & Response Format

Request Headers

All requests must include:

X-API-Key: your-api-key-here
Content-Type: application/json
Accept: application/json

Response Structure

All successful responses follow this structure:

{
  "success": true,
  "message": "Operation successful message",
  "data": { ... }
}

Paginated Responses

List endpoints return paginated data:

{
  "success": true,
  "message": "Data retrieved successfully",
  "data": [ ... ],
  "pagination": {
    "total": 100,
    "page": 1,
    "limit": 10,
    "totalPages": 10
  }
}

HTTP Status Codes

Code Meaning
200 OK - Request successful
201 Created - Resource created successfully
204 No Content - Resource deleted successfully
400 Bad Request - Invalid request parameters
401 Unauthorized - Missing or invalid API key
403 Forbidden - API key lacks required permissions
404 Not Found - Resource not found
405 Method Not Allowed - HTTP method not supported
409 Conflict - Resource conflict (e.g., cannot delete customer with vehicles)
429 Too Many Requests - Rate limit exceeded
500 Internal Server Error - Server error

6. Error Handling

Error responses follow a consistent format:

{
  "success": false,
  "error": "Error message describing what went wrong",
  "code": 400
}

Common Error Scenarios

401 Unauthorized
Missing or invalid API key. Ensure the X-API-Key header is included and valid.
403 Forbidden
API key doesn't have the required permissions. Check if your key has read/write/admin permissions.
400 Bad Request
Invalid request parameters or missing required fields. Check the request body and query parameters.
404 Not Found
The requested resource doesn't exist. Verify the resource ID or endpoint path.
429 Too Many Requests
Rate limit exceeded. Wait before making additional requests or upgrade your API key limits.

7. Rate Limits

API rate limits are configured per API key. Default limits are:

  • Standard Keys: 1000 requests per hour
  • Premium Keys: 10,000 requests per hour
  • Enterprise Keys: Custom limits

Rate limit information is included in response headers:

X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 950
X-RateLimit-Reset: 1640995200
‚ö†Ô∏è Rate Limit Exceeded: When rate limits are exceeded, you'll receive a 429 status code. Implement exponential backoff in your application to handle rate limits gracefully.

import requests

API_KEY = 'your-api-key-here'
BASE_URL = 'https://api.gtvmotor.dev/api/v1'

headers = {
    'X-API-Key': API_KEY,
    'Content-Type': 'application/json'
}

# Get customers list
def get_customers(page=1, limit=10):
    url = f'{BASE_URL}/customers'
    params = {'page': page, 'limit': limit}
    response = requests.get(url, headers=headers, params=params)
    response.raise_for_status()
    return response.json()

# Get customer by ID
def get_customer(customer_id):
    url = f'{BASE_URL}/customers/{customer_id}'
    response = requests.get(url, headers=headers)
    response.raise_for_status()
    return response.json()

# Create customer
def create_customer(customer_data):
    url = f'{BASE_URL}/customers'
    response = requests.post(url, headers=headers, json=customer_data)
    response.raise_for_status()
    return response.json()

# Usage
try:

    print('Customers:', customers)
except requests.exceptions.RequestException as e:

import 'dart:convert';
import 'package:http/http.dart' as http;

class GTVMotorAPI {
  static const String apiKey = 'your-api-key-here';
  static const String baseUrl = 'https://api.gtvmotor.dev/api/v1';
  
  static Map<String, String> get _headers => {
    'X-API-Key': apiKey,
    'Content-Type': 'application/json',
    'Accept': 'application/json',
  };

  // Get customers list
  static Future<Map<String, dynamic>> getCustomers({
    int page = 1,
    int limit = 10,
    String? search,
  }) async {
    try {
      final queryParams = {
        'page': page.toString(),
        'limit': limit.toString(),
        if (search != null) 'search': search,
      };
      
      final uri = Uri.parse('$baseUrl/customers').replace(
        queryParameters: queryParams,
      );
      
      final response = await http.get(uri, headers: _headers);
      
      if (response.statusCode == 200) {
        return json.decode(response.body);
      } else {
        throw Exception('Failed to load customers: ${response.statusCode}');
      }
    } catch (e) {
      throw Exception('Error fetching customers: $e');
    }
  }

  // Get customer by ID
  static Future<Map<String, dynamic>> getCustomer(int id) async {
    try {
      final uri = Uri.parse('$baseUrl/customers/$id');
      final response = await http.get(uri, headers: _headers);
      
      if (response.statusCode == 200) {
        return json.decode(response.body);
      } else if (response.statusCode == 404) {
        throw Exception('Customer not found');
      } else {
        throw Exception('Failed to load customer: ${response.statusCode}');
      }
    } catch (e) {
      throw Exception('Error fetching customer: $e');
    }
  }

  // Create customer
  static Future<Map<String, dynamic>> createCustomer({
    required String name,
    required String phone,
    String? email,
    String? address,
  }) async {
    try {
      final uri = Uri.parse('$baseUrl/customers');
      final body = json.encode({
        'name': name,
        'phone': phone,
        if (email != null) 'email': email,
        if (address != null) 'address': address,
      });
      
      final response = await http.post(
        uri,
        headers: _headers,
        body: body,
      );
      
      if (response.statusCode == 201 || response.statusCode == 200) {
        return json.decode(response.body);
      } else {
        final error = json.decode(response.body);
        throw Exception(error['error'] ?? 'Failed to create customer');
      }
    } catch (e) {
      throw Exception('Error creating customer: $e');
    }
  }

  // Update customer
  static Future<Map<String, dynamic>> updateCustomer(
    int id, {
    String? name,
    String? phone,
    String? email,
    String? address,
  }) async {
    try {
      final uri = Uri.parse('$baseUrl/customers/$id');
      final body = json.encode({
        if (name != null) 'name': name,
        if (phone != null) 'phone': phone,
        if (email != null) 'email': email,
        if (address != null) 'address': address,
      });
      
      final response = await http.put(
        uri,
        headers: _headers,
        body: body,
      );
      
      if (response.statusCode == 200) {
        return json.decode(response.body);
      } else {
        final error = json.decode(response.body);
        throw Exception(error['error'] ?? 'Failed to update customer');
      }
    } catch (e) {
      throw Exception('Error updating customer: $e');
    }
  }

  // Delete customer
  static Future<void> deleteCustomer(int id) async {
    try {
      final uri = Uri.parse('$baseUrl/customers/$id');
      final response = await http.delete(uri, headers: _headers);
      
      if (response.statusCode != 204 && response.statusCode != 200) {
        final error = json.decode(response.body);
        throw Exception(error['error'] ?? 'Failed to delete customer');
      }
    } catch (e) {
      throw Exception('Error deleting customer: $e');
    }
  }

  // Get vehicles list
  static Future<Map<String, dynamic>> getVehicles({
    int page = 1,
    int limit = 10,
    String? search,
  }) async {
    try {
      final queryParams = {
        'page': page.toString(),
        'limit': limit.toString(),
        if (search != null) 'search': search,
      };
      
      final uri = Uri.parse('$baseUrl/vehicles').replace(
        queryParameters: queryParams,
      );
      
      final response = await http.get(uri, headers: _headers);
      
      if (response.statusCode == 200) {
        return json.decode(response.body);
      } else {
        throw Exception('Failed to load vehicles: ${response.statusCode}');
      }
    } catch (e) {
      throw Exception('Error fetching vehicles: $e');
    }
  }

  // Get vehicle by ID
  static Future<Map<String, dynamic>> getVehicle(int id) async {
    try {
      final uri = Uri.parse('$baseUrl/vehicles/$id');
      final response = await http.get(uri, headers: _headers);
      
      if (response.statusCode == 200) {
        return json.decode(response.body);
      } else {
        throw Exception('Failed to load vehicle: ${response.statusCode}');
      }
    } catch (e) {
      throw Exception('Error fetching vehicle: $e');
    }
  }

  // Get analytics
  static Future<Map<String, dynamic>> getAnalytics({
    String type = 'overview',
    int days = 7,
    String? breakdown,
    String? endpoint,
  }) async {
    try {
      final queryParams = {
        'type': type,
        'days': days.toString(),
        if (breakdown != null) 'breakdown': breakdown,
        if (endpoint != null) 'endpoint': endpoint,
      };
      
      final uri = Uri.parse('$baseUrl/analytics').replace(
        queryParameters: queryParams,
      );
      
      final response = await http.get(uri, headers: _headers);
      
      if (response.statusCode == 200) {
        return json.decode(response.body);
      } else {
        throw Exception('Failed to load analytics: ${response.statusCode}');
      }

      throw Exception('Error fetching analytics: $e');
    }
  }
}

Usage Example in Flutter Widget:

import 'package:flutter/material.dart';

class CustomersPage extends StatefulWidget {
  @override
  _CustomersPageState createState() => _CustomersPageState();
}

class _CustomersPageState extends State<CustomersPage> {
  List<dynamic> customers = [];
  bool isLoading = true;
  String? error;

  @override
  void initState() {
    super.initState();
    _loadCustomers();
  }

  Future<void> _loadCustomers() async {
    setState(() {
      isLoading = true;
      error = null;
    });

    try {
      final response = await GTVMotorAPI.getCustomers(page: 1, limit: 10);
      setState(() {
        customers = response['data'] ?? [];
        isLoading = false;
      });
    } catch (e) {
      setState(() {
        error = e.toString();
        isLoading = false;
      });
    }
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Customers'),
      ),
      body: isLoading
          ? Center(child: CircularProgressIndicator())
          : error != null
              ? Center(
                  child: Column(
                    mainAxisAlignment: MainAxisAlignment.center,
                    children: [
                      Text('Error: $error'),
                      SizedBox(height: 16),
                      ElevatedButton(
                        onPressed: _loadCustomers,
                        child: Text('Retry'),
                      ),
                    ],
                  ),
                )
              : ListView.builder(
                  itemCount: customers.length,
                  itemBuilder: (context, index) {
                    final customer = customers[index];
                    return ListTile(
                      title: Text(customer['name'] ?? ''),
                      subtitle: Text(customer['phone'] ?? ''),
                      trailing: Text('${customer['vehicle_count']} vehicles'),
                      onTap: () async {
                        try {
                          final customerData = await GTVMotorAPI.getCustomer(
                            customer['id'],
                          );
                          // Navigate to customer details page
                          Navigator.push(
                            context,
                            MaterialPageRoute(
                              builder: (context) => CustomerDetailsPage(
                                customerData: customerData['data'],
                              ),
                            ),
                          );
                        } catch (e) {
                          ScaffoldMessenger.of(context).showSnackBar(
                            SnackBar(content: Text('Error: $e')),
                          );
                        }
                      },
                    );
                  },
                ),
    );
  }
}

PHP

<?php
$apiKey = 'your-api-key-here';
$baseUrl = 'https://api.gtvmotor.dev/api/v1';

// Get customers list
function getCustomers($page = 1, $limit = 10) {
    global $apiKey, $baseUrl;
    
    $url = $baseUrl . '/customers?page=' . $page . '&limit=' . $limit;
    $ch = curl_init($url);
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
    curl_setopt($ch, CURLOPT_HTTPHEADER, [
        'X-API-Key: ' . $apiKey,
        'Content-Type: application/json'
    ]);
    
    $response = curl_exec($ch);
    $httpCode = curl_getinfo($ch, CURLINFO_HTTP_CODE);
    curl_close($ch);
    
    if ($httpCode !== 200) {
        throw new Exception('HTTP error: ' . $httpCode);
    }
    
    return json_decode($response, true);
}

// Get customer by ID
function getCustomer($id) {
    global $apiKey, $baseUrl;
    
    $url = $baseUrl . '/customers/' . $id;
    $ch = curl_init($url);
